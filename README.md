# ECE 3400 — Intelligent Physical Systems
**Cornell University, Fall 2018**

## Project Overview
A semester-long systems engineering project building an autonomous robot capable of navigating and mapping a maze in real time. The robot integrated IR detection, PID control, and wireless communication between an Arduino microcontroller and an FPGA base station.

Our team placed **3rd in the final competition**, and the project website was awarded **Best Website** in the class.

## Live Site
[View the project website](https://katherinejlu.github.io/ece3400/)

## Technical Highlights
- **PID controller** for line-following and navigation
- **IR detection** for wall sensing and maze traversal
- **Arduino → FPGA communication** for real-time maze mapping and data transmission
- **Verilog** for FPGA logic design
- **C++** for Arduino firmware
- Full project documentation website built with **HTML, CSS, Bootstrap, and JavaScript**

## Repository Structure
- `Final Robot Code/RobotFSM` — final Arduino/C++ state machine for robot control
